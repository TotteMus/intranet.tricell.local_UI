# User Interface Design

## Overview

This repository contains an example of a user interface for a web application.

It is a student project within the university course **"User Interface Design" (5 ECTS)** offered by Åland University of Applied Sciences.

The purpose is to teach first-year university students how to design and implement a modern, responsive user interface for a web application.

This project builds upon the web application developed in the two previous courses, **"Frontend Development" (5 ECTS)** and **"Backend
Development" (5 ECTS)**, making this course a natural continuation of earlier work.

This repository only consists of the new frontend. Students must integrate it with their own backend created in the previous course.

---

## Project Structure

/config          – Configuration files
/data            – Data sources in various formats for exercises (JSON, XML, MDB)
/masterframe     – Frontend HTML files
/public/css      – Stylesheet files
/public/images   – Layout images (GIF, JPG, PNG)
/public/photos   – Personnel registry images
/public/scripts  – Frontend JavaScript files
/routes          – Route definitions (JavaScript)
/                – Main application files
